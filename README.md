# DateDivideProgressBar
等分的日期展示进度条

说明：
* 等分的时间段，每个时间段的天数可以不同，输入日期可以定位到相关的位置上。
* 输入的日期格式为yyyy-MM-dd。
* 控件比较简单，仅自定义控件和时间计算两个类，主要的部分就是对进度和圆圈位置的计算。
* 没有支持xml中的自定义样式，相关的样式可以通过类中的变量控制。
* 不支持拖动，仅是一个展示进度的功能。
